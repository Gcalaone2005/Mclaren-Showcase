## 🚗 McLaren Supercar Showcase

A fully responsive and interactive web showcase of McLaren's iconic supercar lineup. Built using **HTML**, **CSS**, **JavaScript**, and **jQuery**, this project dynamically displays detailed information, specifications, and high-resolution images for each model.
