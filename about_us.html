McLaren Supercar Showcase

This project is a dynamic, responsive website showcasing McLaren’s most iconic supercars. Built with HTML, CSS, JavaScript, and jQuery, it displays detailed specs, images, and animations for each model.

Features

  • Display of multiple McLaren models
  • Car specs: engine, horsepower, acceleration, top speed
  • "More Info" expandable details for each model
  • Hover effects and responsive design
  • Uses JavaScript arrays to dynamically render content

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • jQuery (for DOM manipulation)
  • CDN-hosted images

How to Use

  • Clone the repository or download the files.
  • Open mclaren_showcase.html in any modern browser.
  • Browse the McLaren lineup and click “More Info” to reveal details.

Folder Structure

  • mclaren_showcase.html – main webpage
  • All CSS and JS is written inline or included in the file
  • No installation needed – just run it in your browser
